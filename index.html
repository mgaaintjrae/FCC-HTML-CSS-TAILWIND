Installation

1

Installez Tailwind via npm

Pour la plupart des projets (et pour profiter des fonctionnalités de personnalisation de Tailwind), vous voudrez installer Tailwind via npm. 
            
              # Using npm
              npm install tailwindcss

              # Using Yarn
              yarn add tailwindcss

2

Ajoutez Tailwind à votre CSS

Utilisez la @tailwind directive pour injecter de Tailwind base, components et les utilities styles dans votre CSS: 
            
              @tailwind base;

              @tailwind components;

              @tailwind utilities;
Tailwind échangera ces directives au moment de la construction avec tous ses CSS générés.

ou

Utilisation de Tailwind via CDN

            
              link href="https://unpkg.com/tailwindcss@^1.0/dist/tailwind.min.css" rel="stylesheet"

Preflight

Tailwind injecte automatiquement ces styles lorsque vous incluez @tailwind basedans votre CSS: 

            
              @tailwind base; /* Preflight will be injected here */

              @tailwind components;

              @tailwind utilities;

Les marges par défaut sont supprimées

Responsive Design

Pour ajouter un utilitaire mais pour qu'il ne prenne effet qu'à un certain point d'arrêt, tout ce que vous avez à faire est de préfixer l'utilitaire avec le nom du point d'arrêt, suivi du :caractère: 

          
            /* Width of 16 by default, 32 on medium screens, and 48 on large screens */
            img class="w-16 md:w-32 lg:w-48" src="..."
          
        

          
            /* Small (sm) */
            @media (min-width: 640px) { /* ... */ }

            /* Medium (md) */
            @media (min-width: 768px) { /* ... */ }

            /* Large (lg) */
            @media (min-width: 1024px) { /* ... */ }

            /* Extra Large (xl) */
            @media (min-width: 1280px) { /* ... */ }

Container

Un composant pour fixer la largeur d'un élément au point d'arrêt actuel.

sm (640px) max-width: 640px;
md (768px) max-width: 768px;
