INSTALLATION

1

Installez Tailwind via npm

Pour la plupart des projets (et pour profiter des fonctionnalités de personnalisation de Tailwind), vous voudrez installer Tailwind via npm. 
            
              # Using npm
              npm install tailwindcss

              # Using Yarn
              yarn add tailwindcss

2

Ajoutez Tailwind à votre CSS

Utilisez la @tailwind directive pour injecter de Tailwind base, components et les utilities styles dans votre CSS: 
            
              @tailwind base;

              @tailwind components;

              @tailwind utilities;
Tailwind échangera ces directives au moment de la construction avec tous ses CSS générés.

ou

Utilisation de Tailwind via CDN

            
              link href="https://unpkg.com/tailwindcss@^1.0/dist/tailwind.min.css" rel="stylesheet"

PREFLIGHT

Tailwind injecte automatiquement ces styles lorsque vous incluez @tailwind basedans votre CSS: 

            
              @tailwind base; /* Preflight will be injected here */

              @tailwind components;

              @tailwind utilities;

Les marges par défaut sont supprimées

RESPONSIVE DESIGN

Pour ajouter un utilitaire mais pour qu'il ne prenne effet qu'à un certain point d'arrêt, tout ce que vous avez à faire est de préfixer l'utilitaire avec le nom du point d'arrêt, suivi du :caractère: 

          
            /* Width of 16 by default, 32 on medium screens, and 48 on large screens */
            img class="w-16 md:w-32 lg:w-48" src="..."
          
        

          
            /* Small (sm) */
            @media (min-width: 640px) { /* ... */ }

            /* Medium (md) */
            @media (min-width: 768px) { /* ... */ }

            /* Large (lg) */
            @media (min-width: 1024px) { /* ... */ }

            /* Extra Large (xl) */
            @media (min-width: 1280px) { /* ... */ }

Overflow

Utilitaires pour contrôler la façon dont un élément gère un contenu trop volumineux pour le conteneur.
Class Properties
.overflow-auto overflow: auto;
.overflow-hidden overflow: hidden;
.overflow-visible overflow: visible;
.overflow-scroll overflow: scroll;
.overflow-x-auto overflow-x: auto;
.overflow-y-auto overflow-y: auto;
.overflow-x-hidden overflow-x: hidden;
.overflow-y-hidden overflow-y: hidden;
.overflow-x-visible overflow-x: visible;
.overflow-y-visible overflow-y: visible;
.overflow-x-scroll overflow-x: scroll;
.overflow-y-scroll overflow-y: scroll;

1. Visible

Permet .overflow-visible d'empêcher le contenu d'un élément d'être écrêté. Notez que tout contenu dépassant les limites de l'élément sera alors visible.

2. Auto

Utilisez .overflow-auto pour ajouter des barres de défilement à un élément dans le cas où son contenu déborde les limites de cet élément. Contrairement à .overflow-scroll ce qui affiche toujours les barres de défilement, cet utilitaire ne les affiche que si le défilement est nécessaire.

3. Hidden

Permet .overflow-hiddende découper tout contenu dans un élément qui déborde les limites de cet élément.

4. Scroll horizontally if needed

Utilisez .overflow-x-autopour permettre le défilement horizontal si nécessaire.

5. Scroll vertically if needed

Utilisez .overflow-y-auto pour autoriser le défilement vertical si nécessaire.

6. Scroll horizontally always

Permet .overflow-x-scrolld'autoriser le défilement horizontal et d'afficher toujours les barres de défilement à moins que les barres de défilement toujours visibles soient désactivées par le système d'exploitation.

7. Scroll vertically always

Permet .overflow-y-scroll d'autoriser le défilement vertical et d'afficher toujours les barres de défilement à moins que les barres de défilement toujours visibles soient désactivées par le système d'exploitation.

8. Scroll in all directions

Utilisez .overflow-scrollpour ajouter des barres de défilement à un élément. Contrairement à .overflow-auto ce qui ne montre les barres de défilement que si elles sont nécessaires, cet utilitaire les affiche toujours. Notez que certains systèmes d'exploitation (comme macOS) masquent les barres de défilement inutiles quel que soit ce paramètre.

9. Momentum-based scrolling on touch devices

Utilisez .scrolling-touch pour utiliser le défilement basé sur l'élan (lorsque cela est pris en charge) sur les appareils tactiles.
Utilisez .scrolling-autopour utiliser le défilement normal non basé sur l'élan sur les appareils tactiles. Ceci est surtout utile pour annuler .scrolling-touchsur des écrans de plus grande taille.

Position

Utilitaires pour contrôler la position d'un élément dans le DOM.

Usage

Position

Class Properties
.static position: static;
.fixed position: fixed;
.absolute position: absolute;
.relative position: relative;
.sticky position: sticky;

1. Static

Permet .static de positionner un élément selon le flux normal du document.
Tout décalage sera ignoré et l'élément n'agira pas comme référence de position pour les enfants positionnés de manière absolue.

2. Relative

Permet .relative de positionner un élément selon le flux normal du document.
Les décalages sont calculés par rapport à la position normale de l'élément et l'élément servira de référence de position pour les enfants positionnés de manière absolue.

3. Absolute

Permet .absolute de positionner un élément en dehors du flux normal du document, ce qui fait que les éléments voisins agissent comme si l'élément n'existait pas.
Les décalages sont calculés par rapport au parent le plus proche qui a une position autre que static, et l'élément servira de référence de position pour les autres enfants positionnés de manière absolue.

4. Fixed

Utilisez .fixedpour positionner un élément par rapport à la fenêtre du navigateur.
Les décalages sont calculés par rapport à la fenêtre et l'élément servira de référence de position pour les enfants positionnés de manière absolue.

5. Sticky

Utilisez .sticky pour positionner un élément relative jusqu'à ce qu'il franchisse un seuil spécifié, puis traitez-le comme fixe jusqu'à ce que son parent soit hors écran.
Les décalages sont calculés par rapport à la position normale de l'élément et l'élément servira de référence de position pour les enfants positionnés de manière absolue.
